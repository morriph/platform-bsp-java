# Platform Java

The Reactive Platform API for Java.

## Building

### Linux/Mac

```bash
git clone https://github.com/reactivemarkets/platform-java
cd platform-java
./gradlew build
```

### Windows

```bash
git clone https://github.com/reactivemarkets/platform-java
cd platform-java
./gradlew.bat build
```
